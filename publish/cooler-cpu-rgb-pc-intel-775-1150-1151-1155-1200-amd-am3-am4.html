# Descrição do produto

Descrição

>>> PROMOÇÃO <<<

O cooler KP-VR330 RGB da KNUP tem 2 heat pipes para uma melhor condução de calor, CPU heat pipe em contato direto para uma melhor condutividade térmica e entrega o fluxo de ar suficiente para dispersar o calor e um design moderno.

Características:
- Modelo: KP-VR330
- Marca: KNUP

Compatibilidade:

- Intel: LGA 775/ 115X/ 1200/ 1700/ 1366
- AMD: AM5/ AM4/ FM2+/ FM2/ FM1/ AM3+/ AM3/ AM2+/ AM2
- Pode ser aplicado para LGA2011, porém os acessórios não incluem o suporte de fixação do LGA2011

Dados técnicos:

- TDP: 90W
- Dimensões: 120x62x140mm
- Dimensões Fan: 120x120x25mm
- Rolamento: Hidráulico
- Velocidade do Fan: 1500RPM (+/-10%)
- Max. Fluxo de ar: 40 CFM
- Ruído: 26 dB(A) 10%
- Tensão Nominal: 12v DC
- Tensão Operacional: 10.8~13.2VDC
- Corrente: 0.23A (+/-10%)
- Entrada de Alimentação: 2.64W
- Dissipador de calor: 100mm(c) x 37mm(L) x 128mm(A) heat pipe (contato direto) + alumínio fino

Itens inclusos:
- 01 x Cooler KP-VR330
- 01 x Base e pinos de instalação para instalação
- 01 x Pasta térmica para 1 aplicação
- 01 x Manual do produto
